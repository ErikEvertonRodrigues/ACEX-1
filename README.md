# Fluxo de Desenvolvimento
Este projeto utiliza um fluxo baseado em branches e Pull Requests para manter a organização do código

## Clonar o projeto
```bash 
git clone <url-do-repositorio>
cd nome-do-projeto
```

## Criar branch 
```bash 
git checkout -b task
```

## Atualizar antes de começar uma task
```bash
git pull
```

## Pull Request
Após finalizar a task:
- Abra um Pull Request para `main`
- Descreva as alterações realizadas
- Aguarde review antes do merge
