1. O que é Git e por que usar?

Introdução

Git é um sistema de controle de versão distribuído criado para facilitar o desenvolvimento de software e o trabalho em equipe.

Com ele é possível acompanhar alterações no código, voltar para versões anteriores e colaborar com outros desenvolvedores sem perder histórico.

Por que usar Git?

  • Controle de versões.
  • Histórico completo de alterações.
  • Trabalho colaborativo.
  • Integração com GitHub.
  • Maior segurança para o projeto.

Nesta aula você entenderá os conceitos fundamentais antes de instalar e configurar o Git em sua máquina.
